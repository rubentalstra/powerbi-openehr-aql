# Signed-cert install (future)

When the project ships releases signed with a real code-signing certificate (EV/OV from Sectigo, DigiCert, or SSL.com), this page will document the trusted-publisher install — no self-signed `.cer` import required.

Today, v0.1.0 uses a self-signed cert. See [install-self-signed.md](install-self-signed.md).
